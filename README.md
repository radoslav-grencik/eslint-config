# Installation

Install the package and its peer dependencies:

```bash
npx install-peerdeps --dev @radoslav-grencik/eslint-config
```

## Configuration

Add to `package.json`:

```json
{
  "eslintConfig": {
    "extends": "@radoslav-grencik/eslint-config"
  },
  "prettier": "@radoslav-grencik/prettier-config"
}
```
