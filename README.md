## Installation

Install the package and its peer dependencies:

```bash
npx install-peerdeps --dev @radoslav-grencik/eslintrc
```

## Configuration

Add to `package.json`:

```json
{
  "eslintConfig": {
    "extends": "@radoslav-grencik/eslintrc"
  },
  "prettier": "@radoslav-grencik/prettierrc"
}
```
